HKDV Tax Calculator
This calculator can be used for smile market, buyer/select shop, etc. It helps you calculate the break-even price, accounting for taxes and desired coin profits.
If you are using this for buyer/select shop, the number that goes here is the original purchase price of the item from buyer shop. For example, if you bought the item for 1000 coins, put 1000 here. If you are using this for smile market, the number that goes here is the amount you want in your gift box. You can put "0" in the "Desired Profit (coins):" section.
If you are using this for buyer/select shop, the number that goes here is always 5%. If you are using this for smile market, put the correct tax rate depending on your member rank (Silver: 9%, Gold: 7%, Platinum: 5%).
If you are using this for buyer/select shop, this is the amount you want to earn after buying the item from buyer shop and listing it for sale in smile market. If you are using this for smile market, put "0" here.

The lowest price you can list your item for in the market is in order to receive in your gift box.
